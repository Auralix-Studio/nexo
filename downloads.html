Descargas

Binarios oficiales por plataforma. Todos los archivos incluyen valor SHA-256 para verificación de integridad.

Android

Sideload de APK firmado. Durante la primera instalación, Android solicitará autorización para instalar aplicaciones de fuentes desconocidas.

Variante Archivo Tamaño Compatibilidad
arm64-v8a nexo-v1.1.1-arm64.apk 159 MB Android desde 2017 Descargar
armeabi-v7a sin Lumen nexo-v1.1.1-armv7.apk 27 MB Dispositivos 32-bit Descargar
x86_64 nexo-v1.1.1-x86_64.apk 82 MB Emuladores Descargar
Universal recomendada nexo-v1.1.1-universal.apk 147 MB Cualquier dispositivo Descargar

Cuando se desconoce la arquitectura del dispositivo, la variante Universal garantiza la compatibilidad a costa de un tamaño mayor.

Windows

Paquete portable con asistente de instalación integrado. Al ejecutar nexo.exe por primera vez, la aplicación presenta dos opciones:

  • Instalar. Copia los archivos a %LOCALAPPDATA%\Nexo, crea accesos directos y se registra en el Panel de control.
  • Modo portable. La aplicación se ejecuta desde la carpeta extraída, sin alterar el sistema.

La instalación no requiere privilegios de administrador.

ArchivoTamañoRequisitos
nexo-v1.1.1-windows-x64.zip 222 MB Windows 10 1809 (x64) Descargar

Los binarios no están firmados con certificado de validación extendida. Si Windows Defender SmartScreen muestra una advertencia, seleccionar "Más información" → "Ejecutar de todas formas".

iOS, macOS y Linux

No existe distribución oficial en esta versión. La aplicación compila para todas estas plataformas. Para obtener un binario, abrir un issue en el repositorio.

Web

Demostración en línea disponible en nexo.upla.dev. La versión web tiene restricciones respecto a las nativas: las notificaciones no están disponibles, la integración con Microsoft Teams requiere proxy CORS, y Lumen descarga el modelo en cada carga del sitio.

Verificación de integridad

Cada archivo publica su valor SHA-256. El usuario puede comparar el valor calculado localmente con el publicado para confirmar que el archivo descargado no ha sido alterado.

ArchivoSHA-256
nexo-v1.1.1-arm64.apkpendiente al publicar el release
nexo-v1.1.1-armv7.apkpendiente al publicar el release
nexo-v1.1.1-x86_64.apkpendiente al publicar el release
nexo-v1.1.1-universal.apk9971a160a587976dbe8bca367a982deab5378ceeae545bd6b32e837c500ce288
nexo-v1.1.1-windows-x64.zippendiente al publicar el release

Comando para PowerShell

Get-FileHash -Algorithm SHA256 .\nexo-v1.1.1-universal.apk

Comando para Bash

sha256sum nexo-v1.1.1-universal.apk

Si el valor obtenido no coincide con el publicado, el archivo está corrupto o ha sido manipulado y no debe instalarse.
