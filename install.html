Instalación

Procedimientos detallados por plataforma. La instalación de Nexo no requiere registros adicionales: las credenciales son las mismas de SIGMA UPLA.

Android

Procedimiento

  1. Acceder al último release del repositorio.
  2. Descargar el APK correspondiente al dispositivo.
  3. Abrir el archivo descargado desde el gestor de archivos.
  4. Android solicitará autorización para instalar aplicaciones de fuentes desconocidas. Activar "Permitir desde esta fuente" y confirmar la instalación.
  5. Abrir Nexo desde el menú de aplicaciones.
  6. Aceptar los términos y autenticarse con las credenciales de SIGMA UPLA.

Windows

Procedimiento

  1. Descargar nexo-v1.1.1-windows-x64.zip desde el último release.
  2. Extraer el contenido del archivo en una carpeta temporal.
  3. Ejecutar nexo.exe desde la carpeta extraída.
  4. En el primer arranque, la aplicación presenta un asistente con dos opciones:
A

Instalar

Nexo se copia a %LOCALAPPDATA%\Nexo, crea accesos directos en el escritorio y en el menú Inicio según la selección, y se registra en el Panel de control.

B

Modo portable

La aplicación se ejecuta desde la carpeta extraída sin copiar archivos a otras ubicaciones del sistema.

Requisitos

  • Windows 10 versión 1809 o superior, arquitectura x64.
  • 4 GB de RAM mínimo. 6 GB recomendados con Lumen activado.
  • 500 MB de espacio para la aplicación. 800 MB adicionales con el modelo Lumen Ligero.

Los binarios no están firmados con certificado de validación extendida. Si Windows Defender SmartScreen presenta una advertencia, seleccionar "Más información" y a continuación "Ejecutar de todas formas".

iOS

La aplicación compila correctamente para iOS, pero no se distribuye oficialmente en App Store. La instalación requiere compilación local con Xcode y sideload mediante una cuenta Apple Developer.

La publicación oficial en App Store está condicionada a la disponibilidad de los fondos necesarios para el programa.

macOS y Linux

No existe distribución oficial actualmente. La aplicación compila para ambas plataformas. Para obtener un binario, abrir un issue en el repositorio público.

Web

Demostración en nexo.upla.dev. Limitaciones de la versión web:

  • Las notificaciones no están disponibles.
  • La integración con Microsoft Teams no se encuentra habilitada.
  • El modelo de Lumen se descarga al cargar el sitio.

Activación de Lumen

Procedimiento común a todas las plataformas:

  1. Localizar el botón flotante de Lumen en la esquina inferior derecha de la pantalla principal.
  2. Tocar el botón. Se abrirá un diálogo de bienvenida que describe la política de privacidad y presenta el selector de variante.
  3. Seleccionar la variante adecuada (Ligero para dispositivos con 2-3 GB de RAM, Estándar para 4 GB o superior).
  4. Confirmar con "Aceptar y descargar". La descarga es única y solo se repite al cambiar de variante.
  5. Al finalizar, el asistente queda disponible para su uso.

El modelo se almacena en el directorio privado de la aplicación. Su eliminación se realiza desde Lumen → Configuración → "Borrar modelo".

Desinstalación

Android

Mantener pulsado el icono de la aplicación y seleccionar "Desinstalar". El sistema elimina automáticamente todos los datos asociados.

Windows

Panel de control → Aplicaciones → seleccionar "Nexo UPLA" → Desinstalar.

Eliminación selectiva del modelo Lumen

Lumen → Configuración → "Borrar modelo". Libera el almacenamiento sin desinstalar la aplicación completa.
